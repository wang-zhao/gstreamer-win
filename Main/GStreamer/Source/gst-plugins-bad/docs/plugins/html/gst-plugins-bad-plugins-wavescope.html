wavescope

wavescope — Simple waveform oscilloscope

Synopsis

Properties

Description

Wavescope is a simple audio visualisation element. It renders the waveforms like on an oscilloscope.

Example launch line

1
 
gst-launch audiotestsrc ! audioconvert ! wavescope ! ximagesink
